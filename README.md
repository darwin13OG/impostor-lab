# 🎭 El Impostor - Juego de Fiesta Web & PWA

Un videojuego web de deducción social y fiesta en formato **No-Install**, optimizado para correr de forma fluida tanto en dispositivos móviles como en computadoras de escritorio desde un único archivo.

🌐 **¡Juega la versión oficial aquí!** -> [impostor-lab.pages.dev](https://impostor-lab.pages.dev)

## 🚀 Filosofía del Proyecto
* **Simplicidad:** Todo el juego se ejecuta desde un archivo `index.html`.
* **Cero Instalación:** Acceso instantáneo directo al juego desde la web a través de Cloudflare Pages.
* **Offline Ready:** Funciona como una PWA (Progressive Web App) ejecutable sin conexión a internet.
* **Experiencia de Usuario (UX):** Interfaz limpia en modo oscuro, libre de anuncios invasivos y optimizada para controles táctiles.

## 🕹️ Mecánica de Juego
1. **Configuración:** Selección de jugadores, modos de juego (Original, Infiltrado, Espía) y categorías de palabras contrastantes.
2. **Reparto Seguro:** Distribución de roles ocultos pasando el dispositivo de mano en mano mediante un pad táctil de retención.
3. **Debate:** Temporizador integrado para la fase de interrogación IRL (En la vida real).
4. **Votación:** Sistema de selección para revelar al infiltrado.

## 🛠️ Requisitos Técnicos Mínimos
* **Hardware:** Dispositivos móviles o PC con un mínimo de 1GB de RAM (Altamente compatible con procesadores como Intel Core i3 o MediaTek Helio).
* **Almacenamiento:** ~30 KB.
* **Compatibilidad:** Navegadores modernos con soporte para HTML5, CSS3 y JS Nativo.
