# 🎭 El Impostor - Juego de Fiesta Web & PWA

Un videojuego web de deducción social y fiesta en formato **No-Install & Zero-Budget**, optimizado para correr de forma fluida tanto en dispositivos móviles como en computadoras de escritorio. Desarrollado con una arquitectura monolítica autocontenida en un único archivo.

## 🚀 Filosofía del Proyecto
* **Simplicidad:** Todo el juego se ejecuta desde un archivo `index.html`.
* **Cero Instalación:** Acceso instantáneo desde la web a través de Cloudflare Pages.
* **Offline Ready:** Funciona como una PWA (Progressive Web App) ejecutable sin conexión a internet.
* **Experiencia de Usuario (UX):** Interfaz limpia en modo oscuro, libre de anuncios invasivos y optimizada para controles táctiles interactivos.

## 🕹️ Mecánica de Juego
1. **Configuración:** Selección de jugadores, modos de juego (Original, Infiltrado, Espía) y categorías de palabras contrastantes.
2. **Reparto Seguro:** Distribución de roles ocultos pasando el dispositivo de mano en mano mediante un pad táctil de retención.
3. **Debate:** Temporizador integrado para la fase de interrogación IRL (In Real Life).
4. **Votación:** Sistema de selección y descarte para revelar al infiltrado.

## 🛠️ Requisitos Técnicos Mínimos
* **Hardware:** Dispositivos móviles o PC con un mínimo de 1GB de RAM (Altamente compatible con procesadores de gama de entrada como Intel Core i3 o MediaTek Helio).
* **Almacenamiento:** ~30 KB.
* **Compatibilidad:** Navegadores modernos con soporte para HTML5, CSS3 y JS Nativo (ES6+).
